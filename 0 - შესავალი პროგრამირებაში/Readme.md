# рЃарЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ?


рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ (Software) рЃљрЃарЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃЮрЃЉрЃџрЃўрЃЮрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃА, рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃА рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃА рЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ, рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃўрЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃЮрЃА, рЃБрЃцрЃарЃЮ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃљрЃЊ рЃЎрЃў рЃфрЃћрЃюрЃбрЃљрЃџрЃБрЃарЃў рЃърЃарЃЮрЃфрЃћрЃАрЃЮрЃарЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃюрЃљрЃЎрЃарЃћрЃЉрЃўрЃљ, рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў РђЊ рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃљрЃърЃћрЃЉрЃў, рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃў, рЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃў рЃЊрЃљ рЃАрЃ«рЃЋрЃљ.
рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў РђЊ рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃў (Windows, macOS, Linux).
рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў РђЊ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃљрЃюрЃбрЃўрЃЋрЃўрЃарЃБрЃАрЃћрЃЉрЃў, рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.


# рЃарЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ?

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљрЃА рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃћрЃЉрЃА (Python, JavaScript, C++, Java рЃЊрЃљ рЃАрЃ«рЃЋ.), рЃарЃљрЃЌрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљрЃю рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЮрЃю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў.

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА:

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃўрЃЊрЃћрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃДрЃљрЃџрЃўрЃЉрЃћрЃЉрЃљ РђЊ рЃарЃљ рЃБрЃюрЃЊрЃљ рЃљрЃЎрЃћрЃЌрЃЮрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃЏ.
рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљ РђЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃўрЃЌ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.
рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ РђЊ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА (Bug-рЃћрЃЉрЃўрЃА) рЃърЃЮрЃЋрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ.
рЃљрЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ РђЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ.


# рЃарЃљ рЃљрЃарЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў?

рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃА рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЋрЃБрЃгрЃЮрЃЊрЃЮрЃЌ рЃФрЃџрЃўрЃћрЃарЃў рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџрЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА, рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃА рЃБрЃЎрЃћрЃЌрЃћрЃЉрЃА. рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃарЃљрЃЋрЃљрЃџ рЃАрЃљрЃ«рЃћрЃА рЃЋрЃ«рЃЋрЃЊрЃћрЃЉрЃўрЃЌ рЃЊрЃдрЃћрЃЋрЃљрЃюрЃЊрЃћрЃџ рЃЊрЃарЃЮрЃерЃў, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃЕрЃЋрЃћрЃюрЃА рЃЏрЃўрЃћрЃа рЃДрЃЮрЃЋрЃћрЃџрЃЊрЃдрЃўрЃБрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў: рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃћрЃЉрЃў,
рЃбрЃљрЃЉрЃџрЃћрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃўрЃърЃћрЃЊрЃћрЃЉрЃўрЃф рЃЏрЃфрЃўрЃарЃћ рЃќрЃЮрЃЏрЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃћрЃЉрЃА рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃћрЃю.



# рЃарЃЮрЃЊрЃўрЃА рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў?
рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃўрЃДрЃЮ ENIAC (Electronic Numerical Integrator and Computer), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃћрЃА рЃ»рЃЮрЃю рЃћрЃЦрЃћрЃарЃбрЃЏрЃљ (John Presper Eckert) рЃЊрЃљ рЃ»рЃЮрЃю рЃЏрЃЮрЃЕрЃџрЃўрЃЏ (John Mauchly) 1945 рЃгрЃћрЃџрЃА, рЃљрЃерЃе-рЃерЃў.

­Ъћ╣ ENIAC-рЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў:

рЃўрЃДрЃћрЃюрЃћрЃЉрЃЊрЃљ рЃЋрЃљрЃЎрЃБрЃБрЃЏрЃБрЃа рЃЏрЃўрЃџрЃћрЃЉрЃА (Vacuum Tubes) рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃџрЃљрЃЊ.
рЃўрЃгрЃЮрЃюрЃўрЃЊрЃљ 30 рЃбрЃЮрЃюрЃљрЃА рЃЊрЃљ рЃўрЃЎрЃљрЃЋрЃћрЃЉрЃЊрЃљ рЃЊрЃўрЃЊ рЃАрЃўрЃЋрЃарЃфрЃћрЃА.



# рЃБрЃцрЃарЃЮ рЃЋрЃарЃфрЃџрЃљрЃЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃЏрЃљрЃЌ рЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљрЃќрЃћ
 ,


рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃћрЃЉрЃў, рЃўрЃЊрЃћрЃБрЃарЃљрЃЊ рЃАрЃљрЃЎрЃЏрЃљрЃЮрЃЊ рЃ░рЃњрЃљрЃЋрЃА рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃћрЃюрЃћрЃЉрЃА, рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ рЃЎрЃў рЃўрЃА рЃљрЃарЃўрЃА рЃарЃЮрЃЏ рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃћрЃЉрЃў рЃљрЃарЃљ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА рЃљрЃарЃљрЃЏрЃћрЃЊ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃА
рЃћрЃџрЃљрЃърЃљрЃарЃљрЃЎрЃћрЃЉрЃўрЃљрЃю, рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃћрЃЉрЃАрЃљрЃф рЃЌрЃљрЃЋрЃўрЃАрЃў рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў рЃЊрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃћрЃЉрЃБрЃарЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃАрЃљрЃџрЃљрЃърЃљрЃарЃљрЃЎрЃЮ рЃћрЃюрЃћрЃЉрЃА.



рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃћрЃЉрЃў рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃБрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃўрЃДрЃЮрЃцрЃљ 3 рЃюрЃљрЃгрЃўрЃџрЃљрЃЊ:
# 1. рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃћрЃюрЃћрЃЉрЃў (Low-Level Languages)
1. рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃћрЃюрЃћрЃЉрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃдрЃАрЃљрЃЦрЃЏрЃћрЃџрЃўрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЎрЃў рЃърЃўрЃарЃўрЃЦрЃўрЃЌ - рЃарЃЌрЃБрЃџрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃцрЃљрЃЦрЃбрЃўрЃБрЃарЃљрЃЊ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ рЃЊрЃљ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃўрЃА рЃЏрЃўрЃћрЃа, рЃ«рЃерЃўрЃарЃљрЃЊ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃў рЃ░рЃњрЃљрЃЋрЃА CPU-рЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃА, рЃЏрЃљрЃњ. рЃљрЃАрЃћрЃЏрЃЉрЃџрЃў
# 2. рЃерЃБрЃљ рЃЊрЃЮрЃюрЃўрЃА рЃћрЃюрЃћрЃЉрЃў (Mid-Level Languages)
2. рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЊрЃЮрЃюрЃўрЃА рЃћрЃюрЃћрЃЉрЃерЃў рЃЎрЃў рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃБрЃЎрЃћрЃЌрЃљрЃљ рЃАрЃљрЃЦрЃЏрЃћ, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃњрЃў рЃЏрЃћрЃбрЃљрЃЊ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃљрЃюрЃЦрЃљрЃюрЃБрЃарЃў рЃЎрЃЮрЃЊрЃўрЃА рЃюрЃўрЃерЃюрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃБрЃџрЃЮрЃЉрЃўрЃА рЃЊрЃЮрЃюрЃћ рЃўрЃЏрЃљрЃбрЃћрЃЉрЃА
# 3. рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃћрЃюрЃћрЃЉрЃў (High-Level Languages)
3. рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃћрЃюрЃћрЃЉрЃерЃў рЃЎрЃў рЃЏрЃљрЃдрЃљрЃџрЃў рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃБрЃџрЃЮрЃЉрЃљрЃљ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃўрЃљ, рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў рЃћрЃюрЃўрЃА рЃЊрЃЮрЃюрЃћрЃќрЃћ рЃгрЃћрЃарЃўрЃљ рЃЊрЃљ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃЏрЃћрЃюрЃбрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃўрЃА рЃЏрЃўрЃћрЃа
