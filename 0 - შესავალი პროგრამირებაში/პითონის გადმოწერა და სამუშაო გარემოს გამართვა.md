# როგორ დავრწმუნდეთ რომ python წარმატებით დაყენდა?

python --version და დავაჭიროთ enter-ს, ამის შემდეგ ჩვენ ვნახავთ წარმატებით დაინსტალირების შემთხვევაში პითონის ვერსია, რაც ნიშნავს რომ
